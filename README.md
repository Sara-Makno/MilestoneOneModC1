# MilestoneOneModC1

This repository contains my weekly Jupyter Notebook work for **Module C: Data Science Capstone**.

## Project Topic

This project focuses on using machine learning to support breast cancer classification. The analysis compares two breast cancer datasets:

1. **Breast Cancer Wisconsin Diagnostic Dataset**
   Contains tumor morphology measurements such as radius, texture, perimeter, area, smoothness, compactness, concavity, and diagnosis.

2. **Breast Cancer Coimbra Dataset**
   Contains patient-level clinical and blood-based biomarkers such as age, BMI, glucose, insulin, HOMA, leptin, adiponectin, resistin, and MCP-1.

The goal is to evaluate how well different machine learning techniques can distinguish between benign and malignant or cancer-positive cases.

## Notebooks Included

This repository includes notebooks for Weeks 1–6:

* Week 1: Linear Regression, Polynomial Terms, and Interaction Terms
* Week 2: Ridge, Lasso, and Elastic Net Regression
* Week 3: Forward Selection, Backward Selection, PCR, and PLSR
* Week 4: Logistic Regression and Feature Scaling
* Week 5: Support Vector Machines, Kernel Trick, and Regularization
* Week 6: Decision Trees and Random Forests
* Combined Notebook: Weeks 1–6 summary notebook
